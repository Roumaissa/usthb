Corriger ce code html pour qu’il soit compatible avec le standard XHTML


Un document xhtml doit être conforme à ces restrictions je teste les modificatios :
  • Les éléments doivent être proprement imbriqués
  • Une page XHTML doit avoir une seule racine
  • Les balises HTML et les noms des attributs doivent être écrites en minuscule
  • Les éléments HTML doivent être toujours fermés
  • Les valeurs des attributs doivent être entre guillemets
